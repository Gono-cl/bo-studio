This page allows you to define and run a Single Objective Optimization (SOO) experiment.
A single objective optimization problem focuses on finding the best possible value (minimum or maximum) of one target function by adjusting one or more input variables within defined limits. When you want to optimaze multiobjective problem see the page Multi Objective Optimization.

This tutorial walks you through a complete single objective optimization campaign step by step.
You will learn how to define variables, set up the optimization, enter experimental results, and interpret the outcome.

5. Run the Optimization Loopïƒ

Once initial data is submitted, the optimizer proposes the Next Experiment Suggestion â€” the next best parameter combination to test.
